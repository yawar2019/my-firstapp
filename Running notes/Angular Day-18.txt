Guard Service?

if you want to provide based on crediantial redirectin
to routes you can take Guard Service

To implement Guard Service three services you can take

1)CAnActivate   used for Current Route
2)CAnActivateChild  used for child Route
3)CAnDeactivate  when we leave on Route to another Route




To Implement Authentication based on User Crediantial first create AppService


AppService will two Methods

SetAuth method
getAuth  method


SetAuth Method will take as a parameter value to set your localStorage With Value


setAuthDetails(value:string)
  {
    localStorage.setItem('ValidCred',value);
  }


getAuth Method will return value from localStorage

  getAuthDetails()
  {
    return localStorage.getItem('ValidCred')?.toString();
  }





Create Login PAge



Two Methods

Login and Logout


Login witll Set Auth Details value using setAuthDetails method


setAuthDetails will set two type of value true or false

for Login True

login()
{
this.apps.setAuthDetails('true')
}


for Logout False 
logOut()
{

this.apps.setAuthDetails('false')
}
